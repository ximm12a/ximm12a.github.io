将AHK当成 快捷键打开 很好用-AutoHotkey

发表于 2023-03-14 更新于 2023-04-13

将AHK当成 快捷键打开 很好用，以下提供我常用AHK的方式

快捷键打开，分成3个打开用途，1软件，2文件夹，3网站。

所以将键盘设3种，方便记忆。

win键+ctrl键 ：打开软件类

win键+alt键 ：打开网站类

win键+shift键 ：打开文件夹类

;;快捷键 软件

将上面的路径地址，换上你软件地址，然后按如下键

win键+ctrl键 + f ：打开火狐浏览器

win键+ctrl键 + e ：打开Everything

;;快捷键 文件夹

win键+alt键 +d ：打开d盘

win键+alt键 +e ：打开e盘

;;快捷键 网站

win键+shift键 +q ：打开qq邮箱

win键+shift键 +w ：打开微信公众号

另外：可用EDGE网站的应用方式，将网站当成软件打开。方便快捷，需要联网。

;;快捷键 win键+shift键 +b ：用百度搜索选中的文字

;;快捷键 用google搜索选中的文字 g

AutoHotkey

AutoHotkey是一种用于自动化计算机操作的脚本语言。 它可以让用户自定义键盘快捷键，鼠标快捷键和其他可编程的操作，以提高工作效率。 它还可以自动化重复性任务，例如打开特定的文件夹，运行特定的程序，甚至可以模拟鼠标和键盘输入。

AutoHotkey还具有广泛的支持和活跃的社区，用户可以在其官方网站或其他社区网站上找到各种脚本和应用程序。

如果您想要自定义自己的计算机操作或自动化重复任务，那么AutoHotkey是一个非常有用的工具。 它易于学习和使用，并且可以大大提高您的生产力。

; 打开 计算器

Run calc
return

; 打开/激活 记事本
#n::
IfWinExist ahk_class Notepad ;当检测已存在笔记本时则激活,而不是打开,否则打开新的
WinActivate ahk_class Notepad
Else
Run notepad
Return

;Return

;win+c 复制路径文件或文件夹
#c::

;;用 CapsLock 键+数字实现指定窗口的隐藏与显示 http://ahkcn.net/thread-5639.html#

OnExit,ExitSub
CapsLock & 1::zWinHide(1)
CapsLock & 2::zWinHide(2)
CapsLock & 3::zWinHide(3)

zWinHide(zKey){
global
if(chs%zKey%){ ;显示程序
wh:=chs%zKey%
WinShow,ahk_id %wh%
chs%zKey%:=
zCount-=1
}else{ ;隐藏程序
MouseGetPos,,,wh
chs%zKey%:=wh
WinHide,ahk_id %wh%
zCount+=1
}
}
ExitSub:
Loop %zCount%
{
wh:=chs%A_Index%
WinShow,ahk_id %wh%
}
ExitApp

;=========================================================
; 按住 Capslock 键 + 鼠标左键 可移动窗口
; 注意: 在按下鼠标按钮后您可以释放 Capslock 或鼠标中键
; 而不需要在整个拖动过程中按住它.
; 此脚本需要 v1.0.25+. https://wyagd001.github.io/zh-cn/docs/scripts/EasyWindowDrag.htm

Capslock & LButton::
CoordMode, Mouse ; 切换到屏幕/绝对坐标。

MouseGetPos, EWD_MouseStartX, EWD_MouseStartY, EWD_MouseWin
WinGetPos, EWD_OriginalPosX, EWD_OriginalPosY,,, ahk_id %EWD_MouseWin%
WinGet, EWD_WinState, MinMax, ahk_id %EWD_MouseWin%
if EWD_WinState = 0 ; 仅在窗口不处于最大化状态时
SetTimer, EWD_WatchMouse, 10 ; 在用户拖动时跟踪鼠标.
return

EWD_WatchMouse:
GetKeyState, EWD_LButtonState, LButton, P
if EWD_LButtonState = U ; 按钮已经释放, 所以拖动结束.
{
SetTimer, EWD_WatchMouse, off
return
}
GetKeyState, EWD_EscapeState, Escape, P
if EWD_EscapeState = D ; 按下了 Escape, 所以取消拖动.
{
SetTimer, EWD_WatchMouse, off
WinMove, ahk_id %EWD_MouseWin%,, %EWD_OriginalPosX%, %EWD_OriginalPosY%
return
}

; 否则, 改变窗口的位置以匹配
; 用户拖动鼠标引起的鼠标坐标变化:
CoordMode, Mouse
MouseGetPos, EWD_MouseX, EWD_MouseY
WinGetPos, EWD_WinX, EWD_WinY,,, ahk_id %EWD_MouseWin%
SetWinDelay, -1 ; 让下面的移动更快速/平滑.

WinMove, ahk_id %EWD_MouseWin%,, EWD_WinX + EWD_MouseX - EWD_MouseStartX, EWD_WinY + EWD_MouseY - EWD_MouseStartY
EWD_MouseStartX := EWD_MouseX ; 为下一次调用此子程序进行更新.
EWD_MouseStartY := EWD_MouseY
return
